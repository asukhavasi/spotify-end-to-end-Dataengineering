# spotify-end-to-end-Dataengineering
Spotify Playlist to AWS database using python and AWS Lambda

## Project  Information:

In this project, I made an ETL pipleline using Spotify API on AWS. This pipeline will fetch data from my Spotify playlist using an API, transform into 3 different tables and load into AWS data store.
